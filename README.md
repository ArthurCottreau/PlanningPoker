# Planning Poker

Planning Poker est une application locale faite en Python destinée à évaluer l'effort et la difficulté associés au développement de certaines fonctionnalités.

## Compilation

Prérequis :
- Python (<= 3.9)
- [Flet](https://github.com/flet-dev/flet)
- [PyInstaller](https://github.com/pyinstaller/pyinstaller)

Pour construire l'application, veuillez suivre les étapes suivantes.

```
git clone https://github.com/ArthurCottreau/PlanningPoker.git
cd PlanningPoker/src
pyinstaller --onefile main.py
```
